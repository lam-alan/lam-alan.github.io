Resources

Learning

Where to go to learn HTML and CSS (or any other programming language)

Codecademy: An online platform that offers free coding classes. A great starting point for beginners.

freeCodeCamp: Similar to Codecademy. Founded by a non-profit organization. Also provides free coding classes.

w3schools: An online handbook containing documentation and tutorials. Useful for looking up syntax and basic concepts

Stack Overflow: A question and answer platform for developers. Useful for getting answers to complicated or specific problems.

Help

Websites to consult if you get stuck

(Programmers just Google what their problem is. These websites are usually the top results, so they should be visited via a Google search rather than directly. E.g. "html center text vertically")

Tools

Programs used to write and maintain code

Visual Studio Code: A light-weight and robust code editor. Useful features include auto-indenting, auto-complete, and color-coding.

GitHub: A service for version control. It's kind of like Google Docs where it keeps track of the changes you made. Developers use it to save their work in case they make a mistake or they lose their work. It can also be used to host websites, like this one. (Which is why the url has "github".)
